栅格化

        
        栅格规则
        公式：pageWidth = (W + R) × N − R
        当前规格：
        W=50px;
        R=20px;
        N=14;
        pageWidth=960px;

        基于以上规则，计算某一个grid的宽度。
        公式：gridN = (W + R) x N − R
        例：grid3 = (50 + 20) x 3 − 20 = 190px
        优点：
        完美切合960px，充分使用可用空间
        和lufax.com系列页面吻合度较好
        缺点：
        灵活度受限：最右侧的grid必须额外添加class="last"，清掉margin-right:20px;属性
